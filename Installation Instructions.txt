Installation Instructions for running Game: 

1. Make sure you are using the latest webplayer plugin from: http://unity3d.com/webplayer/ and that you are also using PC or Mac.
2. Download ForkKnightWebFinal.zip file
3. Unzip File
4. Open .html file in folder
5. Game should open in your default browser or program

OR

Installation Instructions for uploading game to your own web server:

1. Make sure you are using the latest webplayer plugin from: http://unity3d.com/webplayer/ and that you are also using PC or Mac.
2. When building a Web Player, Unity automatically generates an HTML file next to the player data file. It contains the default HTML code to load the web player data file. Your HTML file is  "yourgame.html". Your player data is "yourgame.unity3d".
3. Upload both .html and.unity3d files to your server

*** You may have to change the directory of unity3d file with in the body of the html doc. So if you have uploaded your .unity3d file to 
"files/themes/yourgame.unity3d" then change the directory in within the body of your html file.


*** For more information about Unity Web Player, visit: 
http://docs.unity3d.com/Documentation/Manual/HTMLcodetoloadUnityWebPlayercontent.html
